# Portal da Turma 3A

Aplicação web da turma com eventos, calendário e recursos.

## Rodar localmente

**Pré-requisito:** Node.js

1. Instale as dependências:
   `npm install`
2. Inicie o servidor de desenvolvimento:
   `npm run dev`
